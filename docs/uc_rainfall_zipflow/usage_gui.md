# 流域雨量グラフ作成 GUI 利用ガイド

このガイドは、`流域雨量グラフ作成（メインウィンドウ）` を使って  
雨量グラフや関連出力を作成する手順をまとめたものです。

## 1. 起動方法

### 推奨（ルートエントリポイント）

```powershell
uv run python run_uc_rainfall_gui.py
```

### 互換（CLI経由）

```powershell
uv run python -m uc_rainfall_zipflow.cli gui
```

## 2. 画面の見方

- 左ペイン: モード選択、入出力、実行設定、実行ボタン
- 右ペイン: 実行ログ、出力一覧、グラフスタイル調整
- 下部ステータスバー: 現在状態（待機中/実行中/完了/失敗）

## 3. 基本手順（解析雨量データモード）

1. `モード` で `解析雨量データ` を選択
2. `入力ZIPディレクトリ` を指定
3. `期間指定方式` を選択
   - `期間を直接指定`: 開始日/終了日を入力
   - `対象日から自動設定（複数可）`: 候補更新して対象日を選択
4. 必要に応じて `グラフ期間`（5日 / 3日前寄せ / 3日中央 / 3日後寄せ）を選択
5. `ポリゴンディレクトリ` と `出力ディレクトリ` を確認
6. `実行設定`（対象流域、出力種別、グラフ指標、SVG出力）を設定
7. `処理を実行` を押す

## 4. 基本手順（Excelデータモード）

1. `モード` で `Excelデータ` を選択
2. `入力Excelファイル` を指定
3. `候補更新` を押してイベント候補を読み込み
4. 対象イベント日（シート）を選択（複数可）
5. `グラフ期間` を選択
6. `出力ディレクトリ` を確認
7. `処理を実行` を押す

注意:
- Excelモードは流域・出力種別が固定のため、GUIからは表示/変更しません。
- 実行前に「対象イベント一覧」の確認ダイアログが表示されます。

## 5. グラフスタイル調整の使い方

- `グラフスタイル調整` を押すと調整ウィンドウを開きます。
- 解析雨量モード:
  - 対象CSVがあれば実データでプレビュー
  - 対象CSVがない場合はテンプレートデータでプレビュー
- Excelモード:
  - 選択中イベントの先頭1件をプレビューに利用
  - 未選択時はテンプレートデータで起動

## 6. よくあるエラーと対処

- `期間エラー（3日 or 5日）`
  - 期間直接指定で、開始日〜終了日が3日または5日になっていません。
- `候補: 0件`
  - 指定フォルダに対象ZIP/Excelシートがない、または形式が想定外です。
- `実行失敗`
  - 右側の `実行ログ` の末尾メッセージを確認してください。
  - 入力パス、日付、対象流域、出力先書き込み権限を再確認してください。

## 7. 補足

- 設定は内部でキャッシュされ、次回起動時に一部入力が復元されます。
- 画面スクリーンショット保存ショートカット:
  - `Ctrl + Shift + S`
  - `F12`
